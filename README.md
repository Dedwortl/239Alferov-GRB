# Уточнение локализации гамма-всплесков методом затенения Земли

Проект посвящён разработке методики уточнения локализации космических гамма-всплесков с помощью анализа затенения неба Землёй. Используются данные с кубсата 239Alferov.

## О проекте

Одиночный кубсат не может определить направление на источник гамма-всплеска. Наш метод использует естественное "экранирование" Земли: если спутник зафиксировал всплеск — источник находится вне области затенения.

## Файлы

- alf_detection.py — обработка кривых блеска
- shadow_radec.py — построение областей затенения в координатах RA/Dec
- shadow_healpix.py — расчёт затенения и визуализация в проекции Mollweide
